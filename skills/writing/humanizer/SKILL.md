# Humanizer: Remove AI Writing Patterns

Use this skill when text sounds statistically generic, over-structured, or emotionally flat.

## Core Workflow

1. identify obvious AI-writing patterns
2. rewrite for rhythm, specificity, and voice
3. preserve the meaning
4. run a final audit for anything that still sounds machine-generated

## Core Rules

- removing AI patterns is not enough; the rewrite still needs a pulse
- vary sentence rhythm
- prefer specific observations over generic abstraction
- keep opinions, uncertainty, and texture where they fit
