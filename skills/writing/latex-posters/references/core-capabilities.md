# Core Capabilities

Use this reference to choose the right LaTeX poster package and layout direction.

## Package Fit

| Package | Best Fit |
|---------|----------|
| `beamerposter` | Familiar Beamer-based academic posters |
| `tikzposter` | Highly customized visual layouts |
| `baposter` | Structured box-based multi-column posters |

## Design Rules

- Favor strong visual hierarchy.
- Keep section flow obvious at a distance.
- Use readable body sizes and high contrast.
- Let figures carry the message; text should support them, not swamp them.
