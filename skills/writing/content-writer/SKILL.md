Create original content, back it with research, and repurpose it across platforms and formats.

## When to Use

- Writing articles, blog posts, or essays
- Creating social media content (X, LinkedIn, TikTok scripts, YouTube)
- Drafting newsletters or one-off email content
- Repurposing existing content for new channels or audiences
- Research-backed content requiring citations and evidence

Use `writing/content-strategy` when the user needs a publishing plan, content pillars, or topic clusters before drafting. Use `writing/email-sequence` when the work is a triggered lifecycle flow rather than a single email.

## Content Creation Workflow

```
1. RESEARCH  → Gather sources, data, expert perspectives
2. OUTLINE   → Structure with hooks, sections, transitions
3. DRAFT     → Write platform-aware content
4. EVIDENCE  → Integrate citations and data points
5. ADAPT     → Repurpose for additional platforms
```

## Platform-Specific Guidelines

### X (Twitter)
- Hook in first 5 words
- One idea per tweet; threads for depth
- Provocative or contrarian angles perform best
- End threads with actionable takeaway

### LinkedIn
- Professional insight with personal angle
- Open with a pattern-interrupt first line
- Use line breaks for readability
- End with a question or CTA

### Newsletter
- Subject line = promise of value
- One core idea per issue
- Scannable structure: headers, bullets, bold key phrases
- Personal voice; reward loyal readers

### Blog / Article
- 1000-2500 words for depth
- SEO-aware headings and structure
- Evidence-backed claims with inline citations
- Clear takeaways; avoid filler

### YouTube / TikTok Scripts
- Hook in first 3 seconds
- Conversational, spoken-word pacing
- Visual cues noted in brackets
- Strong CTA at the end

## Research-Backed Writing

### Finding Evidence
1. Identify 3-5 key claims that need support
2. Search for: peer-reviewed studies, industry reports, expert quotes, data sets
3. Evaluate source credibility (recency, authority, methodology)
4. Note contradicting evidence — address it rather than ignore it

### Integrating Research
- Weave evidence into narrative, don't dump it
- Lead with the insight, cite the source
- Use data to surprise or challenge assumptions
- Balance statistics with storytelling

### Citation Styles

**Inline**: Studies show 40% improvement (McKinsey, 2024).
**Numbered**: Studies show 40% improvement [1].
**Footnote**: Studies show 40% improvement^1

Choose based on audience expectation and platform norms.

## Content Repurposing

### Adaptation Matrix

| Source → Target | Key Changes |
|----------------|-------------|
| Article → Thread | Extract 5-7 key insights, write each as standalone tweet |
| Article → Newsletter | Summarize + add personal angle + exclusive insight |
| Article → LinkedIn | Pull one contrarian take, expand with professional context |
| Podcast → Article | Structure around key quotes, add written context |
| Talk → Blog | Expand outline, add code/examples impossible in talk format |
| Long → Short | Identify single strongest point, lead with it |

### Repurposing Workflow
1. Identify the core insight (1 sentence)
2. List platform-specific constraints (length, tone, format)
3. Adapt structure and voice — don't just truncate
4. Add platform-native elements (hashtags, visuals, CTAs)
5. Cross-link back to the original

### Audience Adaptation
- **Technical → General**: Replace jargon, add analogies, focus on outcomes
- **General → Technical**: Add implementation details, code, architecture
- **Internal → External**: Remove proprietary context, add background

## Quality Checklist

- [ ] Core message is clear in one sentence
- [ ] Hook captures attention in first line/paragraph
- [ ] Evidence supports key claims
- [ ] Structure matches platform conventions
- [ ] Voice is consistent throughout
- [ ] CTA or takeaway is actionable
- [ ] Sources are credited appropriately
