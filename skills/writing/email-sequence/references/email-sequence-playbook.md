# Email Sequence Playbook

Use this reference when tuning pacing, email roles, and copy quality.

## Core Principles

1. One email, one job
2. Value before ask
3. Relevance over volume
4. Clear next step in every message

## Timing Guidance

- Welcome: immediate first email, then one to two days apart early
- Onboarding: concentrated in the first week, then slower follow-up
- Nurture: two to four days apart
- Re-engagement: fewer emails with clearer escalation
- Long gaps reduce momentum; daily blasts create fatigue

## Sequence Archetypes

| Type | Typical length | Main goal |
|------|----------------|-----------|
| Welcome | 3-5 emails | confirm value and orient the subscriber |
| Onboarding | 4-7 emails | drive the user to the first meaningful outcome |
| Nurture | 4-8 emails | build trust and prepare conversion |
| Re-engagement | 3-4 emails | recover attention or clean the list |
| Launch | 2-5 emails | create awareness, urgency, and follow-through |

## Subject And Preview Guidance

- clear beats clever
- benefit beats vagueness
- preview text should extend the subject, not repeat it
- avoid subject lines that feel like clickbait if the email itself is plain

## CTA Guidance

- one primary CTA per email
- match the CTA to the stage of the relationship
- buttons for main actions, text links for secondary paths
- prefer action plus outcome wording over generic commands

Examples:
- Start your setup
- See the template
- Book the walkthrough
- Return to your workspace

## Copy Checks

- short paragraphs
- obvious white space
- conversational tone
- enough context for the ask to make sense
- no hard sell before the user has received real value
