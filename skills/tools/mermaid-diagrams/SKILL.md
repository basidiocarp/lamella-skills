# Mermaid Diagramming

Use this skill when the task is to visualize structure, flow, architecture, or data relationships in Mermaid.

## Diagram Selection

| Need | Diagram |
|------|---------|
| domain model | class diagram |
| request or message flow | sequence diagram |
| process or decision logic | flowchart |
| database schema | ERD |
| architecture | C4 |

## Core Workflow

1. choose the diagram type that matches the question
2. draft the smallest diagram that answers it
3. validate syntax early
4. add styling or layout only after the semantics are correct
