# Muda (Waste) Analysis Examples

Use these examples to translate the classic seven wastes into software work.

## Example: API Codebase Review

```markdown
SCOPE: REST API backend

OVERPRODUCTION
- Unused endpoints
- Overbuilt abstractions

WAITING
- Slow build/test cycle
- Review bottlenecks

DEFECTS
- Repeat incidents tied to weak test coverage
```

## The Seven Wastes in Software

1. Overproduction: building more than is needed
2. Waiting: blocked work and idle queues
3. Transportation: unnecessary hops or handoffs
4. Over-processing: work with no customer or delivery value
5. Inventory: too much unfinished work
6. Motion: context switching and avoidable task movement
7. Defects: rework, bugs, and incident-driven churn
