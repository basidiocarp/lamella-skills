# Handoff

Create structured session handoff documents for continuity across sessions.

## Use This When

- ending a work session
- switching projects
- taking a break mid-task
- preparing for a known reset or compaction

## Capture

- current task and progress
- decisions and why they were made
- changed files and intent
- blockers and open questions
- clear next steps

## Resume Pattern

Start the next session with the handoff file and ask to resume from it.
