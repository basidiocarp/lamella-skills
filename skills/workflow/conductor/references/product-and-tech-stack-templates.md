# Product and Tech Stack Templates

Use these starters when creating the core project-context artifacts.

## `product.md`

```markdown
# [Product Name]

## Problem
- Who has the problem?
- What pain or friction exists today?

## Outcome
- What changes when the product works?
```

## `tech-stack.md`

```markdown
# Tech Stack

| Technology | Purpose | Source of Truth |
|------------|---------|-----------------|
| TypeScript | App code | package.json |
| PostgreSQL | Primary data store | infra docs |
```
