# Messaging Pattern Templates

## Task Assignment

```
You've been assigned task #{id}: {subject}.

Owned files:
- {file1}
- {file2}

Key requirements:
- {requirement1}
- {requirement2}

Interface contract:
- Import {types} from {shared-file}
- Export {types} for {other-teammate}

Let me know if you have questions or blockers.
```

## Integration Point Notification

```
My side of the {interface-name} interface is complete.

Exported from {file}:
- {function/type 1}
- {function/type 2}

You can now import these. The contract matches what we agreed on.
```

## Blocker Report

```
Blocked on task #{id}: {subject}.

Blocker: {description}
Impact: {what can't be completed}

Options:
1. {option 1}
2. {option 2}

Waiting for guidance.
```

## Task Completion

```
Task #{id} complete: {subject}

Changes:
- {file1}: {what changed}
- {file2}: {what changed}

Integration notes:
- {considerations for other teammates}

Ready for next assignment.
```

## Review/Investigation Summary

```
{Review/Investigation} complete for {target}.

Verdict/Summary: {result}
Confidence: {if applicable}

Key findings:
- {finding 1}
- {finding 2}

Full report attached to task #{id}.
```
