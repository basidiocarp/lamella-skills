# Triggering Examples

Use this page as the routing layer for authoring `<example>` blocks in agent
descriptions.

## Load Order

| Need | Reference |
| --- | --- |
| example block structure and anatomy | `trigger-example-structure.md` |
| example types and coverage strategy | `trigger-example-types.md` |
| common mistakes and corrections | `trigger-example-mistakes.md` |

## Core Rules

- examples should teach when an agent triggers, not restate the whole prompt
- vary phrasing enough to show intent coverage without bloating the file
- commentary should explain the decision boundary clearly
