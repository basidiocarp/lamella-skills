# Agent and Skill Organization

Agents and skills should be grouped by the way users and Claude discover them.

## Agent Patterns

- role-based
- capability-based
- workflow-stage-based

Choose one main organizing principle per plugin so the agent set feels
intentional instead of accidental.

## Skill Patterns

- topic-based
- tool-based
- workflow-based

Keep `SKILL.md` focused on routing and decision support. Put heavier detail into
adjacent references, examples, scripts, or assets.
