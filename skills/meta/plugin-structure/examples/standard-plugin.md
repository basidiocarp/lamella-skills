# Standard Plugin Example

Use this example when you want a balanced plugin with commands, agents, skills, and hooks but do not need enterprise-level MCP or deep shared-library structure.

## What It Demonstrates

- a complete `.claude-plugin/plugin.json`
- a small command set
- a couple of focused agents
- one skill with adjacent references
- hook-based validation
