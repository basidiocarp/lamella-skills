# Weekly Review Process

The weekly review turns accumulated observations into concrete skill updates.

## Steps

1. check whether a review is due
2. load all open observations
3. inventory current skills
4. map observations to:
   - existing skill updates
   - new skills
   - declines
5. apply updates
6. mark processed observations `ACTIONED`
7. update the review timestamp
8. present a short summary

## Summary Format

```markdown
## Weekly Review Summary

**Observations reviewed:** [count]
**Skills updated:** [count]
**New skills created:** [count]
**Declined:** [count]
```
