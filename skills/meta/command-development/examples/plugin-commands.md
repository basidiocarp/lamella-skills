# Plugin Command Examples

Use these examples when commands need to call plugin-local scripts, templates, agents, or skills through portable `${CLAUDE_PLUGIN_ROOT}` paths.

## Covered Patterns

- script-backed commands
- template-backed generation
- multi-step release or deploy flows
- commands that delegate to plugin agents or skills
