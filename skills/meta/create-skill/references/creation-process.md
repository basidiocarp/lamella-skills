# Skill Creation Process

Step-by-step guide for creating a new skill from scratch.

## Workflow

1. understand the skill through concrete examples
2. identify reusable scripts, references, or assets
3. create the skill directory and `SKILL.md`
4. implement bundled resources first
5. write or refine `SKILL.md`
6. validate the skill
7. iterate based on real use

## Minimum `SKILL.md`

```markdown
---
name: skill-name
description: Use when [trigger] - [what the skill does]
---

## Purpose
[short purpose]

## When to Use
[trigger guidance]

## How to Use
[workflow and resource references]
```
