# Color Systems

Use this page as the routing layer for color-system design.

## Load Order

| Need | Reference |
| --- | --- |
| palette construction and perceptual scales | `palette-construction.md` |
| semantic theming and dark-mode token mapping | `semantic-color-tokens.md` |
| contrast, accessibility, and color-vision checks | `contrast-and-color-accessibility.md` |

## Core Rules

- separate palette generation from semantic role assignment
- test theme mappings in real UI combinations, not only swatch files
- keep accessibility checks close to token validation
