# Loading Animations

Choose the loading pattern that matches uncertainty and duration.

## Common Patterns

- spinner for short indeterminate waits
- dots or pulse for conversational or inline waits
- skeleton for layout-preserving content loading
- progress bar when work has measurable advancement

## Rules

- prefer the simplest animation that communicates state clearly
- keep motion subtle and readable
- provide reduced-motion fallbacks
- avoid mixing many loading styles in the same flow without a reason
