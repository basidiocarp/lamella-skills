# Command Evaluation Examples

Use these examples when evaluating a command or agent workflow end to end.

## Refactor Command Example

| Criterion | Weight |
|-----------|--------|
| Correctness | 0.40 |
| Completeness | 0.25 |
| Style | 0.20 |
| Efficiency | 0.15 |

## Iteration Loop

1. Identify the weak criterion.
2. Change the prompt or workflow to target that weakness.
3. Re-run the same tests.
4. Confirm that the weak area improved without causing regressions elsewhere.
