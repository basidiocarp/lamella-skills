# Structured Evaluation

Use this reference when the rubric is explicit and most criteria are objective.

## Workflow

1. Define criteria and weights.
2. Build test cases at multiple complexity levels.
3. Run direct scoring against the rubric.
4. Aggregate weighted scores and summarize failures.

## Example Criterion

```markdown
### Criterion: Instruction Following (weight: 0.30)
- 5: All instructions followed precisely
- 3: Major instructions followed, smaller ones missed
- 1: Task fundamentally misunderstood
```
