# Regression and Bias Workflows

Use this reference when comparing variants or tracking evaluation quality over time.

## Position Bias Control

1. Compare A vs B
2. Compare B vs A
3. If results disagree, treat as unstable rather than forcing a winner

## Regression Loop

1. Keep the same test set
2. Change one prompt or workflow variable
3. Re-score
4. Check gains and regressions by criterion
