# Hierarchical and Panel Evaluation

Use this reference when a single scoring pass is not reliable enough.

## Hierarchical Workflow

`Quick screen -> detailed evaluation -> human review if confidence is low`

## Panel Workflow

- Run multiple independent judges with distinct framings.
- Aggregate by median or majority.
- Flag high-variance criteria for follow-up review.

## Best Fit

- High-stakes prompt or workflow changes
- Subjective quality dimensions
- Evaluations where model variance matters
