# Thought-Based Reasoning Techniques

Use these techniques when the task benefits from explicit intermediate reasoning.

## Main Modes

- Chain-of-thought with exemplars
- Zero-shot chain-of-thought
- Self-check or answer-extraction second pass

## When to Use

- Multi-step reasoning
- Ambiguous planning tasks
- Problem decomposition before acting

## Guardrail

Do not force explicit reasoning on simple execution tasks where a direct instruction is clearer and cheaper.
