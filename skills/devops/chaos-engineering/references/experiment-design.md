# Chaos Experiment Design

Use this reference when framing an experiment before choosing the fault mechanism.

## Core Shape

1. State the steady state.
2. Form a falsifiable hypothesis.
3. Define blast radius and time limit.
4. Verify rollback and safety triggers.
5. Decide what evidence counts as success or failure.

## Quick Checklist

- Hypothesis is specific
- Metrics are observable
- Blast radius is constrained
- Rollback is rehearsed
- Learnings will be captured after the run
