# Development Workflow

This file extends [common/git-workflow.md](./git-workflow.md) with the work that happens before git operations.

## Workflow

1. search for an existing implementation or prior art before writing net-new code
2. plan before editing when the task is non-trivial
3. prefer test-first or regression-first changes when behavior is changing
4. run review or self-review before closing higher-risk work
5. verify the actual claim before committing or pushing
