# Writing Clearly and Concisely

Write for fast comprehension.

## Rules

- use active voice and concrete language
- prefer direct statements over hedge-heavy prose
- omit filler, throat clearing, and repeated framing
- keep one paragraph or list item focused on one idea
- choose examples that clarify a real decision, not ones that teach basic writing class
