# Claude Configuration Templates

Templates for configuring Claude Code projects.

## Templates

| File | Description |
|------|-------------|
| `CLAUDE-DNA.md` | Core Claude personality and behavior definition |
| `CLAUDE-MD-GUIDE.md` | Guide for writing CLAUDE.md files |
| `CLAUDE-MD-TEMPLATE.md` | Starter template for CLAUDE.md |
| `PROMPT-GUIDE.md` | Guide for effective prompting |

## Usage

Copy templates to your project and customize:

```bash
cp resources/templates/claude/CLAUDE-MD-TEMPLATE.md .claude/CLAUDE.md
```
